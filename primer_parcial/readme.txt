Primer proyecto evaluativo

Requisitos del proyecto:
- Programación en C#
- Siete (7) interfaces gráficas con siete elementos
- Un procedimiento de cálculo que no sea realizar pagos
- Informe sobre el funcionamiento del programa

** Extraer .zip y ejecutar .exe
** Para visualizar código ejecutar .sln

Descripción: sistema de logística que incluye interfaces con instancias de inicio, registro, verificación de paradas, temporizador que calcula la velocidad del transporte y atención al cliente en caso de fallas.

Nuevos conocimientos: variables globales y locales, if-else, sistema de validaciones al completar campos, navegación entre interfaces, migración de datos de una interfaz a la otra, entre otros.
